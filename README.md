# cat_fighting_instruments
An instrument that plays cat fights using the Raspberry Pi.  
I saw [this video](https://www.youtube.com/watch?v=wjJUMHvj_Jk) and wanted to make it.

## Environments
- Raspberry pi4 model B
- Python3.7
- supercollider3.7
